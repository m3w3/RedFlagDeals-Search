# RedFlagDeals-Search
Search for unique forum posts on RedFlagDeals.

# Setting Up Selenium
Download the Python binding for Selenium:
```Python
pip install selenium
```

Get the latest ChromeDriver here: https://sites.google.com/a/chromium.org/chromedriver/downloads
